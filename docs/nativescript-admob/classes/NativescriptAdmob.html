isAdmobInitialized$

isAdmobInitialized$: BehaviorSubject<boolean>

Overrides NativescriptAdmobCommon.isAdmobInitialized$

  • Defined in packages/nativescript-admob/index.d.ts:37

Emits true when Google Admob has been successfully initialized and is ready to serve ads. Emits false until it is ready.

example
const admob = new NativescriptAdmob();

// Subscribe to the Observable to get the status:
admob.isAdmobInitialized$.subscribe({
  next: (isInitialized) => {
    if (isInitialized) {
      console.log('Google Admob has been successfully initialized!');
    } else {
      console.log('Google Admob has not been initialized yet.');
    }
  },
});

// Or get the current status imperatively:
const isInitialized = admob.isAdmobInitialized$.value;
if (isInitialized) {
  console.log('Google Admob has been successfully initialized!');
} else {
  console.log('Google Admob has not been initialized yet.');
}
