Optional adaptiveBannerWidth

adaptiveBannerWidth?: number
  • Defined in packages/nativescript-admob/index.d.ts:195

ONLY for AD_SIZE.ADAPTIVE_BANNER The width for an adaptive type banner. If not specified, or less than or equal to 0, the width will be the full width of the screen.

remarks
  • The adaptive banner size is designed to work best when using the full available width, so do not use this option if you want to take advantage of this.
  • The size returned for a given width on a given device will always be the same, hence once you've tested your layout on a given device, you can be sure that the ad size will not change.
  • Adaptive banner height is never larger than 15% of the device's height and never smaller than 50 dp.

Optional margins

margins?: { bottom?: number; top?: number }
  • Defined in packages/nativescript-admob/index.d.ts:204

The number of pixels from the top/bottom of the view natural position of this banner size (type). The plugin corrects for display density, so don't worry about that.

If both are set, top wins.

Type declaration
