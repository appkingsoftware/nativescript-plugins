Interface BannerAdCallbacks

Optional callbacks for various events of banner ads.

remarks

These callbacks are optional for each ad, you can implement all, none, or as many as you like.

Optional onAdClicked

  • onAdClicked(): void
    • Defined in packages/nativescript-admob/index.d.ts:294

    Called when a click is recorded for an ad.

    remarks

    Android doesn't call onAdClicked for banner ads (other than Google native ads, which this plugin doesn't support yet), so for pseudo feature parity with iOS, the onAdOpened callback is used to fire this onAdClicked callback. In other words, on Android, the onAdClicked and onAdOpened callbacks are the same, and fire at the same time.

    Returns void
